# ППО калкулатор — Раздел I

Уеб калкулатор за **разчет на силите и средствата за гасене на пожар** при твърди горими
вещества (гасене с вода), по Методиката от **Приложение № 5** — формули 1–23.

Резултатът се извежда като **подробен разчет** (формула → заместени стойности → резултат),
форматиран за копиране директно в MS Word.

## Какво изчислява

По зададени изходни данни:

- **Време** — до съобщението, за движение и за разгръщане
- **Помещение и пожар** — ширина `a`, дължина `b`, линейна скорост на разпространение `Vл`
  (Табл. № 1) и интензивност на подаване на вода `Iн` (Табл. № 2)
- **Гасене** — тип ръчен струйник, брой струйници за защита, дебит на помпата `Qпа`
- **Форма на гасене** — по фронт (едно-/дву-/тристранно), по периметър (правоъгълник/кръг)
  или по цялата площ

калкулаторът пресмята и показва стъпка по стъпка:

- свободно време за развитие на пожара `tсв`
- площ на пожара `Fп` и радиус `Rп`
- площ на гасене `Fг`
- необходим разход на вода `Qн`
- брой струйници, екипи, личен състав и пожарни автомобили

Всяка илюстрация на форма на гасене се отваря в прозорец с чертеж, формула и описание.

## Стартиране

Приложението е **една самостоятелна страница** на чист HTML + JavaScript — без изграждане,
без зависимости, без сървър.

```sh
open index.html
```

Работи директно от `file://`, както и през локален уеб сървър (напр. MAMP/Apache).

## Структура

| Файл / папка | Предназначение |
|---|---|
| `index.html` | Цялото приложение — форма, стилове, изходен панел, модален прозорец |
| `functions.js` | Изчислително ядро и рендиране на разчета |
| `images/` | Схеми на формите на гасене |

## Методика

Формулите следват **Приложение № 5, формули 1–23** — Раздел I: гасене на твърди горими
вещества с вода. `π` се приема за `3.14` съгласно указанията. Калкулаторът работи само с
**ръчни струйници** (дълбочина на гасене `hг = 5 m`).
